# 📝 اپلیکیشن لیست کارها (Todo List) - نسخه دسکتاپ

**بسم الله الرحمن الرحیم**

یک اپلیکیشن دسکتاپ ساده، سبک و کاملاً رایگان برای مدیریت کارهای روزمره و شخصی. 
این برنامه با هدف سادگی و دوری از پیچیدگی‌های اضافی نوشته شده تا بدون حواس‌پرتی، فقط روی انجام کارهایتان تمرکز کنید.

## ✨ ویژگی‌ها
- 🚀 **سبک و سریع:** نوشته شده با Electron، بدون مصرف منابع اضافی سیستم.
- 📴 **کاملاً آفلاین:** نیازی به اینترنت ندارد؛ اطلاعات شما روی سیستم خودتان امن می‌ماند.
- 🎯 **رابط کاربری ساده:** طراحی مینیمال و بدون پیچیدگی، مخصوص کارهای شخصی.
- 💾 **ذخیره‌سازی محلی:** کارهای شما به صورت لوکال ذخیره می‌شوند.
- 🆓 **کاملاً رایگان و آزاد.**
